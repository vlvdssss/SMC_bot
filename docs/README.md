## Терминология

**На русском**: Все описания, объяснения, инструкции

**На английском**: Торговые термины SMC
- BOS (Break of Structure)
- CHoCH (Change of Character)
- OB (Order Block)
- FVG (Fair Value Gap)
- HTF (Higher Time Frame)
- LTF (Lower Time Frame)
- R (Risk units)
- DD (Drawdown)
- WR (Win Rate)
