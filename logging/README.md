# Логирование

## Описание

Система логирования для SMC-framework.

Логи используются для:
- Отладки стратегий
- Мониторинга бэктестов
- Анализа ошибок
- Аудита сделок (для live trading)

## Структура

```
logging/
├── logger.py       # Настройка логирования
└── README.md       # Этот файл
```

## Уровни логирования

- **DEBUG**: Детальная информация для отладки
- **INFO**: Общие информационные сообщения
- **WARNING**: Предупреждения (не критичные проблемы)
- **ERROR**: Ошибки выполнения
- **CRITICAL**: Критические ошибки (требуют немедленного внимания)

## Формат логов

Стандартный формат:
```
[2025-12-17 15:30:45] [INFO] [XAUUSD] Сигнал: BUY на 2850.50
```

Структура:
- Дата/время
- Уровень
- Инструмент (если применимо)
- Сообщение

## Использование

```python
from logging.logger import setup_logger

logger = setup_logger("xauusd_strategy")
logger.info("Анализ H1 завершен")
logger.warning("Низкая ликвидность")
logger.error("Ошибка подключения к MT5")
```
