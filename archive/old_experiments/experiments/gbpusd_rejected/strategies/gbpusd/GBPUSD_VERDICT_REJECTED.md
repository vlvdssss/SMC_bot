## Причины отклонения

### 1. Отсутствие сигналов

**Тестирование 2023**:
- Период: 01.01.2023 - 31.12.2023 (полный год)
- Данные: 6,208 H1 баров, 24,826 M15 баров
- **Результат: 0 сделок**

**Попытки исправления**:
1. ❌ Смягчение параметров (range 20-150 pips вместо 30-80)
2. ❌ Уменьшение rejection критериев (40% вместо 50%)
3. ❌ Упрощение sweep buffer (3 pips вместо 5)
4. ❌ Уменьшение RR requirement (1.0 вместо 1.2)
5. ❌ Полное упрощение range detection (просто recent high/low)

**Итог**: Даже с максимально мягкими параметрами **0 сигналов за год**.

### 2. Фундаментальная проблема концепции

**Mean Reversion логика не работает на GBPUSD**:
- Liquidity sweep detection слишком редкий
- Rejection criteria не соответствуют реальному поведению GBPUSD
- Range-bound assumptions не подтверждаются данными

**GBPUSD в реальности**:
- Не показывает чёткие equal highs/lows для range trading
- Sweeps происходят но rejection недостаточно выражены
- Behaviour больше похож на trending, чем на ranging

### 3. Отсутствие диверсификации

**Оригинальная цель**: Добавить mean reversion компонент в портфель

**Реальность**: 
- Если стратегия не генерирует сигналы = нет диверсификации
- 0% вклад в портфель = бесполезна
- Добавление мёртвого инструмента ухудшит управление

## Lessons Learned

1. **Mean reversion требует специфичных условий**:
   - Чёткие range boundaries
   - Частые sweeps с rejection
   - GBPUSD не показывает этого behaviour

2. **Не каждый инструмент подходит**:
   - GBPUSD trending больше чем ranging
   - Forex pairs нужно тестировать индивидуально
   - Концепция важнее чем желание добавить 3-й инструмент

3. **0 сигналов = красный флаг**:
   - Если стратегия не торгует = она не работает
   - Нельзя оптимизировать пустоту
   - Лучше 2 рабочих инструмента чем 3 с одним мёртвым
