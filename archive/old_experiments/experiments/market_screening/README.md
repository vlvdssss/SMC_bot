# Market Screening Experiment

**Дата**: 20 декабря 2025  
**Статус**: ARCHIVED (незавершённый эксперимент)

## Цель

Fast screening нескольких инструментов для определения market-fit с SMC фреймворком.

## Инструменты

1. USDCHF
2. EURGBP
3. NZDUSD
4. USDJPY
5. AUDCAD
6. XAGUSD

## Подход

- Единая логика SMC Retracement (без адаптации)
- HTF: H1 BOS direction filter
- LTF: M15 OB retracement entries
- Risk: 0.5%, RR: 2:1

## Результаты

**USDCHF** (2023):
- Trades: 8
- WR: 50%
- ROI: 2%
- PF: 1.98
- Verdict: Слабый результат

**Остальные**: Эксперимент прерван (технические проблемы + медленная работа)

## Причина архивации

1. Процесс backtesting слишком медленный (73k+ баров на инструмент)
2. Технические ошибки в индексации
3. Первые результаты показали слабую производительность
4. Решение: архивировать и вернуться к рабочему baseline

## Файлы

- `screening_strategy.py` - Единая стратегия для всех инструментов
- `run_screening.py` - Runner для backtesting
- `download_screening_data.py` - Скрипт загрузки данных
- `data/` - Исторические данные 6 инструментов (H1 + M15, 2023-2025)
- `results/` - Частичные результаты

## Выводы

- Не все инструменты подходят для SMC подхода
- Единая логика без адаптации даёт слабые результаты
- Screening требует более быстрого подхода (меньше данных или оптимизация кода)

## Рекомендации

Если вернёмся к screening:
1. Использовать меньший период (например, только 2024 год)
2. Оптимизировать код стратегии (векторизация pandas)
3. Или тестировать по одному инструменту с более детальным анализом
