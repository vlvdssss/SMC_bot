## Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/vlvdssss/SMC_bot.git
   cd SMC_bot
   ```

2. **Set up environment:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run tests:**
   ```bash
   python main.py --mode backtest --year 2024
   ```

4. **Make changes and test**
